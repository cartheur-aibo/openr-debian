# First project: Hello Body

Goal: prove that Debian -> OPEN-R build -> Memory Stick -> ERS-7 boot works.

## Behavior

1. On startup, set a visible LED/face state.
2. Listen for a simple touch event.
3. On touch, perform a small head nod or emit a diagnostic message.
4. Return immediately to the message loop.

## Done criteria

- The robot boots from the development stick.
- Your object is loaded.
- A visible output changes when the object starts.
- One sensor-driven reaction works.
